# Task Manager (Уровень 1 - Junior)

Полнофункциональное веб-приложение для управления задачами.

## Стек технологий
- **Frontend:** React.js
- **Backend:** Node.js (Express), PostgreSQL
- **Скрипты:** Python

---

## Инструкция по локальному запуску

### 1. Подготовка базы данных
1. Запустите PostgreSQL.
2. Создайте пустую базу данных с именем `task_db`.
3. Выполните SQL-запрос из файла `database/init.sql` для создания таблицы.

### 2. Запуск Backend
1. Перейдите в папку бэкенда: `cd backend`
2. Создайте файл `.env` на основе `.env.example` и впишите пароль от вашей БД.
3. Запустите сервер: `npm start` (Сервер запустится на порту 5000).

### 3. Запуск Frontend
1. В новом окне терминала перейдите в папку фронтенда: `cd frontend`
2. Создайте файл `.env` на основе `.env.example` (оставьте как есть, если бэкенд на порту 5000).
3. Запустите клиент: `npm start` (Приложение откроется на порту 3000).

### 4. Экспорт задач (Python)
1. Перейдите в папку скриптов: `cd scripts`
2. Установите зависимости: `pip install psycopg2 python-dotenv`
3. Создайте файл `.env` на основе `.env.example` и укажите доступы к БД.
4. Запустите скрипт: `python export_tasks.py` (Сгенерируется файл `tasks_export.csv`).
