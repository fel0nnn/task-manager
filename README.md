# Task Manager Project

Full-stack приложение для управления задачами. Разработано с использованием React, Node.js, PostgreSQL и Python.

## Структура проекта и описание файлов

* **`docker-compose.yml`** — конфигурация для автоматического развертывания всех сервисов (БД, Backend, Frontend) в изолированных контейнерах.
* **`database/init.sql`** — SQL-скрипт, который автоматически создает таблицу `tasks` при первом запуске контейнера с базой данных.
* **`backend/`** — серверная часть на Node.js.
    * `server.js` — точка входа, инициализация Express-сервера и маршрутизация REST API (CRUD операции).
    * `db.js` — конфигурация пула подключений к PostgreSQL.
    * `Dockerfile` — инструкции для сборки образа бэкенда.
* **`frontend/`** — клиентская часть на React.
    * `src/App.js` — основной компонент интерфейса и логика взаимодействия с API.
    * `src/App.css` — стили пользовательского интерфейса.
    * `Dockerfile` — инструкции для сборки образа фронтенда.
* **`scripts/export_tasks.py`** — скрипт для подключения к БД и выгрузки всех задач в файл `tasks_export.csv`.
